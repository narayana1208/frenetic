This is the INSTALL file for the frenetic distribution.

This package uses OASIS to generate its build system. See section OASIS for
full information.

Dependencies
============

In order to compile this package, you will need:
* ocaml (>= 4.01.0) for all, test all_tests
* findlib
* cstruct for library lwt, library netcore
* dprle for library lwt, library netcore
* core for library lwt, library netcore, executable distfrenetic,
  executable disttest
* sexplib for library lwt, library netcore
* lwt for library lwt, library netcore, executable katnetic,
  executable netkat_test
* ocamlgraph for library lwt, library netcore
* openflow for library lwt, library netcore
* packet for library lwt, library netcore, executable testtool
* topology for library lwt, library netcore
* quickcheck for library quickcheck, executable testtool
* oUnit for executable testtool
* pa_ounit for executable testtool
* yojson for executable distfrenetic, executable disttest
* async for executable distfrenetic, executable disttest
* async_parallel for executable distfrenetic, executable disttest
* cohttp for executable distfrenetic

Installing
==========

1. Uncompress the source archive and go to the root of the package
2. Run 'ocaml setup.ml -configure'
3. Run 'ocaml setup.ml -build'
4. Run 'ocaml setup.ml -install'

Uninstalling
============

1. Go to the root of the package
2. Run 'ocaml setup.ml -uninstall'

OASIS
=====

OASIS is a program that generates a setup.ml file using a simple '_oasis'
configuration file. The generated setup only depends on the standard OCaml
installation: no additional library is required.
